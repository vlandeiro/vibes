# vibes

A scratchpad for vibe-coded projects — quick experiments and ideas built with an AI coding partner.

Each project lives here while it's being iterated on. When something proves its worth, it gets promoted to its own repository.

## Projects

- **hammerspoon-whisper** — Voice dictation integration for macOS via Hammerspoon and Whisper
- **agent-configuration** — A selective memory system for AI agents using a graph of markdown files with wikilinks, allowing agents to load only the context they need for each task
- **formation-builder** — Interactive sports formation builder for visualizing and managing team lineups
